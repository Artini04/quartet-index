# Vocabulary Index Web App for Japanese Quartet

Vocabulary index web app for Japanese Quartet written in Vue and Nuxt.

## Environment

Using [Nuxt4](https://nuxt.com) and [bunjs](https://bun.sh).

## Running the project

1. Clone the repository to your local machine.
2. Run `bun install` to install dependencies.
3. Run `bun [run] dev` to start the development server. Add `--public` if you need to expose.
4. Open `localhost:3000` to view the web app.
